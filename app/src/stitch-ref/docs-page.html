Boilr is a C++ powered high-performance CLI tool designed for instant scaffolding of modern full-stack applications with integrated cloud infrastructure.

download

Installation

Run this command in your terminal to install the latest version of Boilr.

$ curl -sSL https://boilr.sh/install | sh
terminal

Command Usage

The boilr CLI follows a simple flag-based structure. Most commands start with an action flag followed by your configuration.

Flag Name Description
-init Initialize Initialize a new project in the current directory.
-TN Template Name Specify which scaffold template to use (e.g., node-server).
-N Project Name The name of the resulting folder/project.
-aws AWS Enable Injects AWS SDK and basic IAM roles into the project.
-pr Production Optimizes the build for production environment.

Examples

Create a new React project with AWS integration:

$ boilr -TN react-app-ts -N my-awesome-app -aws

Initialize a minimal Node server for production:

$ boilr -TN node-server-ts -N api-backend -pr
